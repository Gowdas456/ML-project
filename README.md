# ML-project
Machine Learning & Geospatial Analytics Project

Tech Stack: Python, Flask, Scikit-learn, Pandas, NumPy, Folium, SerpApi, Random Forest Algorithm

Overview: A smart retail analysis system developed using Machine Learning and Geospatial Intelligence to help businesses identify profitable retail store locations. The project analyzes important business factors such as population density, income levels, foot traffic, competitor count, and rental cost to generate a predictive Success Score for retail site selection.

How it Works: The system collects and processes real-time geospatial and economic data, then uses a Random Forest Regressor model to predict the business success probability of a retail location. The prediction results are normalized into a 0–100% Success Score and visualized on an interactive Folium map using Green, Orange, and Red zones to represent high, medium, and low business potential areas. The application also integrates SerpApi for competitor analysis and Flask for building an interactive web-based dashboard.
